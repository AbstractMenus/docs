## Авто-перезагрузка

Обычно для применения изменений в файлах меню используется команда `/am reload`. AbstractMenus умеет делать это сам: введите `/am serve`, и плагин начнёт следить за папкой `menus` и автоматически перезагружать изменённые или новые меню. Действие срабатывает при каждом создании или сохранении файла внутри `menus`.

Чтобы выключить - введите `/am serve` ещё раз.

:::caution
Использовать только во время активной разработки меню. На боевом сервере не включать.
:::
